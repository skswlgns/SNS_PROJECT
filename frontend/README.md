### frontend 실행전 설치해야할 목록입니다.

#### npm 설치

```
npm install
```

#### 회원가입 시 주소 검색

```
npm install --save vue-daum-postcode
```

#### 로그인시 cookie
```
npm install --save vue-cookies vuex
```

#### 서버와 통신
```
npm install --save axios
```

#### bootstrap 사용
```
npm install vue bootstrap-vue bootstrap
```
